# **App Name**: Vantage Ledger

## Core Features:

- Unified Entity Management: Comprehensive UI for registering and managing both suppliers and customers for streamlined transactional mapping.
- Institutional Bulk JSON Processor: A high-performance module designed to parse and load purchase data directly from uploaded JSON payloads into the system.
- AI Data Mapper Tool: An intelligent tool that uses reasoning to dynamically identify and map varying JSON invoice keys to the standard financial schema, ensuring data consistency.
- Gain & Cost Analytics Dashboard: Real-time visualization of total earnings, cost structures, and percentage-based profit margins based on processed invoices.
- Persistent Transaction Ledger: Secure storage and retrieval for all uploaded purchase and invoice records using a centralized database architecture.
- JSON Schema Validation: Automated validation for all JSON inputs to prevent formatting errors and data corruption during bulk imports.

## Style Guidelines:

- Primary color: Sophisticated Indigo (#7c3aed) representing logic and structural integrity.
- Background: A deep, midnight charcoal (#12111a) providing a dark, high-contrast environment for data-heavy views.
- Accent color: Vibrant Electric Blue (#3b82f6) used for highlights and specific action-driven financial signals.
- Headline and body font pairing: 'Space Grotesk' for modern, tech-forward headers and 'Inter' for highly legible body and data display. A sans-serif pairing for a neutral, machined look.
- Code display font: 'Source Code Pro' (monospace) for reviewing and editing JSON inputs.
- An institutional, modular grid system that allows for split-pane views—one side for data upload and the other for real-time visualization.
- Precise micro-interactions, such as sequential element loading and progress tracers during JSON parsing phases.

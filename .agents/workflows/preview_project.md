# Preview Project

Follow these steps to start the Next.js development server:

1. **Start Development Server**
   Run the dev script. This will start the Turbopack dev server on port 9002 (as defined in package.json).
   ```bash
   cmd /c "npm run dev"
   ```

2. **Access Preview**
   Once the server starts, open your browser and navigate to `http://localhost:9002` to view and test the application.
